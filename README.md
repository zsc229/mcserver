# zkw-MC-Launcher

一款专为 Minecraft 服务器管理员打造的“零门槛”智能启动器。容易报毒

🌟 核心亮点：内置 AI 智能诊断 | 自动安装 Java 环境 | 商业级账号积分系统 | 密码加密 | 一键下载核心 | Tab 指令补全

📖 项目简介
zkw-MC-Launcher 不仅仅是一个启动器，它是一个完整的服务器运营解决方案。
无论你是刚入门的新手，还是拥有数百名玩家的服主，它都能帮你解决开服路上的所有痛点：

❌ 不再担心环境配置：自动检测17/21。
❌ 不再害怕英文报错：内置 AI 助手，实时监控日志，自动翻译并给出解决方案。
💰 独特的运营模式：内置账号系统与积分经济体系，支持签到、续费、权限管理，适合多服联合运营。
✨ 核心功能特性
🤖 AI 智能助手
实时日志监控：后台自动扫描日志，发现 OutOfMemory、Port Occupied 等致命错误时主动预警。
智能诊断报告：一键生成中文诊断报告，不仅告诉你“错在哪”，还告诉你“怎么修”。
交互式问答：内置知识库，支持自然语言提问（如：“怎么给 OP？”、“服务器卡怎么办？”）。
☕ 自动化环境管理
Java 自动检测：启动时自动扫描系统环境与本地运行时。
一键安装 Java：缺失环境时，自动从镜像源下载并配置内置 Java，无需用户手动操作。
核心一键下载：集成 PaperMC, Purpur, Mojang 官方源，支持断点续传与进度显示。
🔐 商业级账号与积分系统
管理系统，支持注册、登录、登出。
积分经济体系：
📅 每日签到：随机获取 100-500 积分。
💰 服务器续费：10 积分 = 1 天 服务器使用权，过期自动禁止启动。
👑 管理员特权：内置超级管理员 (admin)，拥有无限积分与永久使用权。
银行级安全：所有密码均采用 加密 存储，杜绝明文泄露风险。
🛠️ 极致的开服体验
💻 Tab 智能补全：支持指令补全与在线玩家名自动联想。
⚡ 常用指令库：内置 OP、Gamemode、Give 等高频指令，一键填入。
📝 日志自动归档：每次启动自动生成带时间戳的日志文件，方便回溯。
